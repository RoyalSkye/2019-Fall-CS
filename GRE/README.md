### The overview of GRE

```
GRE是自适应考试，一共6个section，1作文(2个各30min)，然后VQVQV或者QVQVQ，其中多出来的一个V或者Q是加试，V30min，Q35min，第一个V或者Q的难度是median+，一般难度系数70左右，如果你答对7题，第二个section会进入median，答对13题，进入hard，否则easy，当然每个模式都有下限和上限，如果你第二个section进入了hard，那你可以放心你的分数不会低的。
考试顺序：
AW(Analytical Writing):先考Issue(30min)，类似于T的Writing的独立写作，不过要求更高，观点更难，可是我考试之前1篇issue都没练，完全是照着T的标准来写的。写400字+吧；之后考Argument(30min)，对给出的argument进行驳斥，挑毛病，同样字数400+；对于作文呢可以不用太准备，3/3.5的成绩就可以，3.5你需要好好看看Argument，背一下模板(首尾段用一下就行)。GitHub分享的AW里有AW的题库和Argument的提纲，考试前一周过一下。
V(Verbal Reasoning):可以说是重中之重，每个section20道题，10道填空10道阅读。备考资料填空用1200机经就够了，多刷几遍，我在考场上遇到了原题可是答案记不住了。阅读由于我没有时间准备所以基本裸考，正确率也惨不忍睹，同样做240篇阅读机经，也会遇到原题。V的合理时间分配一般是填空10-12分钟，剩下的给阅读，可是我太菜了，基本给填空15分钟，阅读放弃了长阅读，蒙了好几个。
Q:可以说是国人的骄傲了，考试之前做几道易错题，把数学的单词过几遍(单词很重要)！满分不是梦。数学千万别提前交卷，有时间多检查几遍，话说二战第二个hard section我差点没剩时间。
```
