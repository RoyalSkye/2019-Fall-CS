### 备考

#### Words
```
从单词看起，我是最不爱背单词的，背GRE单词的时候每天痛不欲生，不过T的单词还是比较简单的，寒假每天看点最后实在看不下去了索性直接刷题，刚开始文章看不懂，不过慢慢的单词也都认识了，这也许就是量变到质变的过程。这里强调一下学科单词在T中真的很重要，无论是在阅读中还是在听力中，不过我也没有特意去背，毕竟刷了这么多题，到后来基本都知道是什么意思了。
```
#### Reading
```
没什么好说的，刷TPO(我记得有人说TPO30之前很简单，可是我之前做20-25好像是这几套，文章巨难！不过题不难)；现在T reading的趋势是文章生词变多，而且很多长难句(这里说一下，如果你先备考GRE看了GRE长难句教程，那T的阅读可以说很简单了)；真实考试中T的阅读会重复2-3年前的题，通常是采用新题+旧题的形势，所以TPO刷完了就刷2016年左右的题；前期保证量，每天3-4篇文章(模拟真实考试，连着做不要休息，通常考试是R/L加试，我5次考试遇到4次R加试，4篇文章)，，然后精读1篇文章(文章重读+翻译，看懂长难句结构，然后每段看完重新做题)，后期感觉阅读没什么问题了每天做1-2篇文章找一下感觉就OK；然后我的做题方法是：看完一段马上做这一段所对应的题。
```
#### Listening
```
可以说是一道坎了，因为听力贯穿speaking和writing，所以一定要啃下这块硬骨头。刚开始听听力的时候痛不欲生，5分钟的lecture根部集中不了注意力！到后面就已经困了，我前3次考试听力完全没听懂，全是蒙的… 练习的话就是精听了，听写跟读什么的之前都试过，太浪费时间了；我是每天先模拟真实考试做1个小时听力(2个section: 2 conversation and 4 lecture)然后精听差不多2conversation+2-3lecture，就是一句一句听，先自己想这句话什么意思，能不能把每个词听出来，听3-4遍听不出来再看原文，差不多每天精听3小时，然后每天在路上的时候听昨天听过的lecture，直接放全文；听力不需要把每个词都听出来，但要明白大体意思，知道文章结构以及ETS出题点(这个挺重要的，做到后期一听到for example, but这类关键字，就知道要竖起耳朵，考点要来了)，另外关于笔记，这个因人而异，不过我建议conversation如果不是类lecture的conversation就不记笔记或少记，lecture记笔记，但是也不要听到什么就记什么，记你认为是考点的地方，记笔记的时候回漏听后一句，所以权衡，速记都很重要，这些都是练出来的。
```
#### Speaking
```
这个开始task12我真的毫无头绪，不过多亏了我同学，建议直接背万能理由，然后网上套即可，T的口语强调流畅，不用特意去练语音语调，但一定要说的流畅。练习的话每天一套口语即可，30分钟做题，然后1个小时左右重新做task3-6，听看原文，重新记笔记，然后照着笔记重新录音，直到这道题你说到满意为止。每次听别人的录音或者看别人写的题解都感觉写的很好，但是人家也是一遍又一遍的改才写出来的，真实考试就是一遍成，所以跟着自己的笔记走，不需要刻意模仿他人。
```
#### Writing
```
其实我最怕的是综合写作，因为3个点每次都能少听点或听不懂点，考试再一紧张就更完了，所以这个就要平时练，基本是一天一篇吧，综合写作有模板，但是要自己总结出一个模板，每次考试前要修改，否则被ETS发现会直接判limited；独立写作要观点充分，字数400+较好，通常是3个观点或2个正观点1个让步，每个观点最好不要说空话，要么有例子，要么有实验结果支持，这个不像GRE要求那么高，例子可以编啊~但是别太离谱，建议每两天练一篇独立写作。
```
